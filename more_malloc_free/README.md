# more_malloc_free

This project covers more dynamic memory allocation in C using malloc, free, and exit.

## Functions

| File | Prototype | Description |
|------|-----------|-------------|
| 0-malloc_checked.c | `void *malloc_checked(unsigned int b)` | Allocates memory, exits with 98 if malloc fails |

## Requirements

- Ubuntu 20.04 LTS
- gcc with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Only `malloc`, `free`, and `exit` are allowed from the standard library
