# malloc_free

This project covers dynamic memory allocation in C using malloc and free.

## Functions

| File | Prototype | Description |
|------|-----------|-------------|
| 0-create_array.c | `char *create_array(unsigned int size, char c)` | Creates an array of chars initialized with a specific char |

## Requirements

- Ubuntu 20.04 LTS
- gcc with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- Only `malloc` and `free` are allowed from the standard library
